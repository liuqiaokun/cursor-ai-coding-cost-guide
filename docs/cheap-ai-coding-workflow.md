# Cheap AI Coding Workflow

Cheap AI coding does not mean using the weakest model for everything. It means using the right model and workflow for each part of the job.

The most expensive AI coding workflow is usually:

- Start a huge task.
- Use the strongest model for everything.
- Lose context when the session stops.
- Restart the same explanation.
- Repeat.

## A Better Workflow

1. Use a strong model to plan.
2. Split the task into small chunks.
3. Use cheaper runs for mechanical changes.
4. Save progress after every checkpoint.
5. Use a strong model again for debugging and review.
6. Keep multiple agents coordinated from one place.

## Cost Predictability

For heavy users, predictable cost matters more than the lowest possible cost. If you know the cost of a day, week, or month of focused AI coding, you can decide whether the workflow is worth it before starting.
